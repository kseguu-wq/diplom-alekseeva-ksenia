# Автотест API для сервиса «Самокат»

Проект содержит автоматизированный тест сценария:
1. Создание заказа.
2. Получение заказа по треку.
3. Проверка, что код ответа при получении равен 200.

## Запуск теста
- Установи зависимости: `pip install pytest requests`
- Запусти тест: `pytest test_order_creation.py -v`

## Данные для подключения
- Базовый URL: `https://11004837-7242-4644-88b6-d32840e6e123.serverhub.praktikum-services.ru`
- Эндпоинты: `/api/v1/orders` (POST), `/api/v1/orders/track` (GET)

## Логирование
При возникновении ошибок смотри логи на сервере:  
`/var/www/backend/logs/error.log`
